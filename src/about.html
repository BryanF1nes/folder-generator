How to generate folders

  • 1 - Select which folder structure you need
  • 2 - Navigate to the 'Job Folder' location
  • 3 - Create a new folder and use the project name
  • 4 - Select the folder you just created and click 'Select Folder'
  • 5 - Let the 'Folder Generator' do the rest!
